# 操作系统配置
## 终端设置代理

### 1\. Windows (PowerShell)

**配置步骤：**

1.  打开配置文件：
    ```powershell
    if (!(Test-Path $PROFILE)) { New-Item -Type File -Path $PROFILE -Force }
    notepad $PROFILE
    ```
2.  粘贴以下内容（保存后重启终端）：

<!-- end list -->

```powershell
# --- Proxy Functions ---
function proxy {
    $addr = "127.0.0.1:7890"  # ⚠️ 修改端口
    $env:HTTP_PROXY = "http://$addr"
    $env:HTTPS_PROXY = "http://$addr"
    $env:NO_PROXY = "localhost,127.0.0.1,::1,192.168.*,10.*"
    
    Write-Host "Proxy ON ($addr)" -ForegroundColor Green
}

function unproxy {
    Remove-Item Env:\HTTP_PROXY -ErrorAction SilentlyContinue
    Remove-Item Env:\HTTPS_PROXY -ErrorAction SilentlyContinue
    Remove-Item Env:\NO_PROXY -ErrorAction SilentlyContinue
    Write-Host "Proxy OFF" -ForegroundColor Yellow
}

# --- Auto Enable (默认开启) ---
proxy
```

### 2\. macOS (Zsh)

macOS 默认使用 Zsh。

**配置步骤：**

1.  编辑配置文件：
    ```bash
    nano ~/.zshrc
    ```
2.  粘贴以下内容（粘贴后按 `Ctrl+O` 保存，`Ctrl+X` 退出）：

<!-- end list -->

```bash
# --- Proxy Functions ---
function proxy() {
    local port="7890"  # ⚠️ 修改端口
    local addr="127.0.0.1"
    
    export http_proxy="http://${addr}:${port}"
    export https_proxy="http://${addr}:${port}"
    export no_proxy="localhost,127.0.0.1,::1,192.168.0.0/16,10.0.0.0/8"

    echo "Proxy ON (http://${addr}:${port})"
}

function unproxy() {
    unset http_proxy
    unset https_proxy
    unset no_proxy
    echo "Proxy OFF"
}

# --- Auto Enable (默认开启) ---
proxy
```

3.  立即生效：
    ```bash
    source ~/.zshrc
    ```

### 3\. Linux (Bash)

大多数 Linux 发行版（Ubuntu/CentOS）默认使用 Bash。

**配置步骤：**

1.  编辑配置文件：
    ```bash
    nano ~/.bashrc
    ```
2.  粘贴以下内容（与 macOS 几乎一致）：

<!-- end list -->

```bash
# --- Proxy Functions ---
function proxy() {
    local port="7890"  # ⚠️ 修改端口
    local addr="127.0.0.1"
    
    export http_proxy="http://${addr}:${port}"
    export https_proxy="http://${addr}:${port}"
    export no_proxy="localhost,127.0.0.1,::1,192.168.0.0/16,10.0.0.0/8"

    echo -e "Proxy ON (http://${addr}:${port})"
}

function unproxy() {
    unset http_proxy
    unset https_proxy
    unset no_proxy
    echo -e "Proxy OFF"
}

# --- Auto Enable (默认开启) ---
proxy
```

3.  立即生效：
    ```bash
    source ~/.bashrc
    ```
